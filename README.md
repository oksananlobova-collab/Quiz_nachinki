# Подбор начинки для торта — Тортодом

Небольшой лендинг с интерактивным квизом: пользователь отвечает на 4 вопроса и получает 3 подходящие начинки с процентом совпадения и персональным описанием.

## Быстрый старт

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npm install
npm start
```

Альтернатива без npm:

```bash
npx serve .
```

## Структура проекта

```
.

├── .gitignore
└── README.md
```

## Что изменить под клиента

| Что | Где |
|-----|-----|
| Название, город, Telegram | `js/config.js` |
| Начинки и «вау»-тексты | `js/data.js` → `fillings` |
| Вопросы квиза | `js/data.js` → `questions` |
| Популярные карточки на лендинге | `js/data.js` → `highlights` |
| Цвета и шрифты | `css/styles.css` → `:root` |

## Как работает подбор

1. Каждый ответ добавляет баллы начинкам по тегам (`+2` за совпадение).
2. Ответ «чего не хочу» снимает баллы (`−4`).
3. В конце показываются 3 начинки с наибольшим счётом.
4. Процент совпадения считается по тегам и относительному score.
5. Кнопка Telegram формирует сообщение с выбранными вариантами.
